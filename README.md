## Development

1. Add and install CompileDaemon:
    ```sh
    go get github.com/githubnemo/CompileDaemon  
    ```
    ```sh
    go install github.com/githubnemo/CompileDaemon  
    ```
2. Starting the server
    ```sh
    compiledaemon --command="./go-template"
    ```
